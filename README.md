# Учёт рабочего времени
Приложение для учёта рабочего времени

## Описание проекта
Настольное Python-приложение для учёта рабочего времени с возможностью экспорта в excel файл с расширением .xlsx.

## В приложении имеются 3 вкладки
- Сотрудники;
- Проекты;
- Задачи.

## Влкадка сотрудники
На данной вкладке доступны кнопки:
- Добавить нового сотрудника - при нажатии на кнопку открывается новое окно, где нужно заполнить поля для создания нового сотрудника, после заполнения нажмите кнопку "ОК";
- Изменить данные сотрудника - необходимо выбрать сотрудника в таблице, после чего при нажатии на кнопку открывается новое окно, где нужно заполнить те поля, которые вы хотели бы изменить, если не заполнять поля, то изменения не произойдут, после заполнения нажмите кнопку "ОК";
- Добавить отработанные часы сотруднику - необходимо выбрать сотрудника в таблице, после чего при нажатии на кнопку открывается новое окно, где нужно заполнить кол-во добавляемых часов, после заполнения нажмите кнопку "ОК";
- Посчитать зарплату выбранному сотруднику - необходимо выбрать сотрудника в таблице, после чего при нажатии на кнопку открывается новое окно, где будет показана зарплата сотрудника, согласно его отработанным часам;
- Получить email выбранного сотрудника - необходимо выбрать сотрудника в таблице, после чего при нажатии на кнопку открывается новое окно, где будет отображён его EMAIL, если EMAIL некорректен, то будет выведена информация об этом;
- Экспорт в Excel - при нажатии на кнопку актуальный срез таблицы сотрудников экспортируется в Excel в формате .xlsx;
- Обновить таблицу сотрудников - при нажатии на кнопку таблица сотрудников актуализируется данными из БД Postgres.

Также на данной вкладке под кнопками доступна таблица сотрудников.

## Влкадка проекты
На данной вкладке доступны кнопки:
- Добавить новый проект - при нажатии на кнопку открывается новое окно, где нужно заполнить поля для создания нового проекта, после заполнения нажмите кнопку "ОК";
- Изменить название проекта - необходимо выбрать проект в таблице, после чего при нажатии на кнопку открывается новое окно, где нужно заполнить поле 'Название проекта', если не заполнять поле, то изменения не произойдут, после заполнения нажмите кнопку "ОК";
- Назначить проекту задачу - необходимо выбрать проект в таблице, после чего при нажатии на кнопку открывается новое окно, где нужно заполнить идентификатор задачи, которую вы хотите добавить к проекту, идентификатор задачи должен быть существующим, иначе вы получите сообщение с предупреждением, после заполнения нажмите кнопку "ОК";
- Вывести прогресс проекта - необходимо выбрать проект в таблице, после чего при нажатии на кнопку открывается новое окно, где будет показан прогресс по проекту, согласно статусам задач, которые привязаны к данному проекту, если к проекту не привязана ни одна задача, то будет отображена соответствующая информация;
- Экспорт в Excel - при нажатии на кнопку актуальный срез таблицы проектов экспортируется в Excel в формате .xlsx;
- Обновить таблицу проектов - при нажатии на кнопку таблица проектов актуализируется данными из БД Postgres.

Также на данной вкладке под кнопками доступна таблица проектов.

## Влкадка задачи
На данной вкладке доступны кнопки:
- Добавить новую задачу - при нажатии на кнопку открывается новое окно, где нужно заполнить поля для создания новой задачи, после заполнения нажмите кнопку "ОК";
- Изменить данные по задаче - необходимо выбрать задачу в таблице, после чего при нажатии на кнопку открывается новое окно, где нужно заполнить поля, которые вы хотели бы изменить, если не заполнять поля, то изменения не произойдут, после заполнения нажмите кнопку "ОК";
- Назначить задачу сотруднику - необходимо выбрать задачу в таблице, после чего при нажатии на кнопку открывается новое окно, где нужно заполнить идентификатор сотрудника, которому вы хотите назначить задачу, идентификатор сотрудника должен быть существующим, иначе вы получите сообщение с предупреждением, после заполнения нажмите кнопку "ОК";
- Завершить задачу - необходимо выбрать задачу в таблице, после чего при нажатии на кнопку меняется статус выбранной задачи на 'Завершено';
- Вернуть задачу в работу - необходимо выбрать задачу в таблице, после чего при нажатии на кнопку меняется статус выбранной задачи на 'В процессе';
- Экспорт в Excel - при нажатии на кнопку актуальный срез таблицы задач экспортируется в Excel в формате .xlsx;
- Обновить таблицу задач - при нажатии на кнопку таблица задач актуализируется данными из БД Postgres.

Также на данной вкладке под кнопками доступна таблица задач.

## Функционал
- Добавление рабочих часов сотрудникам;
- Хранение данных - в БД Postgres;
- Экспорт данных - в Excel;
- GUI Tk.
